## Requirements

iOS 8.0

## Installation

MediaMathSDK is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

    pod 'MediaMathSDK', :git => 'https://github.com/mmacdermaid/sdkpublic.git'
